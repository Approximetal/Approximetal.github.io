# approximetal.github.io

Personal website of [Zhiyuan Zhao](https://approximetal.github.io/), based on the Renovamen Astro template.

## Commands

| Command      | Action                                      |
| :----------- | :------------------------------------------ |
| `pnpm i`     | Install dependencies                        |
| `pnpm dev`   | Start local dev server at `localhost:4321`  |
| `pnpm build` | Build production site to `./dist/`          |
| `pnpm test`  | Run tests                                   |

## Deployment

Push this repository to:

```text
https://github.com/Approximetal/Approximetal.github.io
```

GitHub Actions will build and deploy the Astro site to GitHub Pages.
